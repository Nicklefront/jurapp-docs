## Root Level

```
jura-lm-app/
├── README.md                        # Project overview
├── pom.xml                          # Maven parent POM (multi-module: desktop + backend)
├── src_pom.xml                      # Standalone desktop fat-JAR build (used by release-desktop.yml)
├── mvnw / mvnw.cmd                  # Maven wrapper
├── docker-compose.yml               # Dev stack (postgres, keycloak, backend)
├── docker-compose.prod.yml          # Production overrides
├── dependency-reduced-pom.xml       # Auto-generated by maven-shade
├── preview-test.sh                  # Manual preview/smoke test script
├── test_backend.html                # Manual backend connectivity test page
│
├── .github/                         # GitHub-specific
│   └── workflows/
│       ├── ci.yml                   # Build + test on push/PR
│       ├── deploy.yml               # SSH-deploy backend to Hetzner on main push
│       ├── release-desktop.yml      # Build DMG/MSI/DEB installers on v* tag
│       └── deploy-pages.yml         # Deploy docs/ to GitHub Pages on docs/** change
│
├── src/                             # Desktop JavaFX application source
├── jurapp-backend/                  # Spring Boot REST backend module
├── docs/                            # Documentation + GitHub Pages site
├── scripts/                         # Operational and developer scripts
├── nginx/                           # nginx reverse-proxy config
├── keycloak/                        # Keycloak realm config
├── deploy/                          # Deployment runbook and helpers
├── package-resources/               # Platform icons and metadata for jpackage
│
├── archive/                         # Old code/experiments (not compiled)
├── backups/                         # Local DB and data backups (gitignored)
├── data/                            # Raw/processed data files (local use)
├── logs/                            # Runtime logs (gitignored)
├── target/                          # Maven build output (gitignored)
└── postgres_data/                   # Local Docker postgres volume (gitignored)
```

## jurapp-backend/ — Spring Boot REST API

```
jurapp-backend/
├── pom.xml                          # Backend module POM
├── Dockerfile                       # Multi-stage: eclipse-temurin:21-jre-alpine
└── src/main/
    ├── java/de/jurapp/backend/
    │   ├── config/                  # Spring @Configuration beans
    │   ├── controller/              # REST endpoints: Auth, Fall, AI, Claims
    │   ├── dto/                     # Request/response DTOs (Jackson)
    │   ├── entity/                  # JPA @Entity classes (ORM)
    │   ├── exception/               # Exception handlers
    │   ├── repository/              # Spring Data JPA repositories
    │   ├── security/                # JWT filter, security config
    │   └── service/                 # Business logic: AI proxy, auth, fall
    └── resources/
        ├── application.yml          # Spring Boot config: DB, JPA, JWT
        └── db/migration/postgresql/ # Flyway SQL for production PostgreSQL
```
